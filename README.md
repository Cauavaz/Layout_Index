# Desafio Front-end - Landing Page Responsiva

## 🎯 Descrição do Projeto

Landing page moderna e responsiva desenvolvida com **HTML5**, **CSS3** e **JavaScript Vanilla** (sem frameworks ou bibliotecas externas), conforme especificações do desafio.

## ✨ Diferenciais Implementados

### 🎨 Design Moderno
- **Gradientes vibrantes** com esquema de cores profissional
- **Glassmorphism** (efeito de vidro fosco) nos cards
- **Animações suaves** e transições fluidas
- **Efeitos de hover** interativos em todos os elementos

### 🚀 Animações Avançadas
- **Floating animation** na imagem principal
- **Glow effects** com pulso animado
- **Fade-in e slide-in** ao carregar a página
- **Bounce effects** nos ícones do carrossel
- **Transições suaves** com cubic-bezier personalizado

### 📱 Responsividade Premium
- **Mobile-first approach**
- **Breakpoints otimizados**: 968px, 640px, 380px
- **Grid responsivo** que se adapta automaticamente
- **Carrossel adaptativo**: 2 itens (desktop) → 1 item (mobile)
- **Tipografia fluida** com clamp() para escalabilidade perfeita

### 🎪 Carrossel Avançado
- **Auto-play inteligente** (pausa ao interagir)
- **Navegação por teclado** (setas esquerda/direita)
- **Swipe touch** para dispositivos móveis
- **Indicadores visuais** com animações
- **Transição infinita** (volta ao início automaticamente)
- **Classe orientada a objetos** para melhor manutenção

### ♿ Acessibilidade
- **ARIA labels** em todos os botões
- **Suporte a teclado** completo
- **Prefers-reduced-motion** para usuários sensíveis a animações
- **Contraste adequado** em todos os textos

### ⚡ Performance
- **CSS otimizado** com variáveis customizadas
- **Animações em GPU** (transform e opacity)
- **Lazy loading** com Intersection Observer
- **Código modular** e bem estruturado

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada com:
  - CSS Grid e Flexbox
  - Custom Properties (variáveis CSS)
  - Gradientes e filtros
  - Animações e transições
  - Media queries
- **JavaScript ES6+** - Funcionalidades interativas:
  - Classes
  - Arrow functions
  - Template literals
  - Event listeners
  - Touch events
  - Intersection Observer API

## 🚀 Como Executar

1. Abra o arquivo `index.html` diretamente no navegador
2. Ou use um servidor local:
   ```bash
   # Com Python 3
   python -m http.server 8000
   
   # Com Node.js (http-server)
   npx http-server
   ```

## 🎯 Funcionalidades do Carrossel

1. **Navegação por botões**: Clique nas setas para navegar
2. **Indicadores**: Clique nos pontos para ir direto ao slide
3. **Auto-play**: Troca automática a cada 4 segundos
4. **Teclado**: Use ← e → para navegar
5. **Touch**: Deslize para os lados em dispositivos móveis
6. **Responsivo**: Adapta quantidade de itens visíveis
